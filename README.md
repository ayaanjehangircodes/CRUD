# React CRUD Dashboard (Admin Panel)

A real-world style **React CRUD Dashboard** that simulates common admin panel functionality such as user management. The project focuses on **core React fundamentals**, clean component architecture, and application logic rather than heavy styling.

---

##  Features

- Display data in a table layout  
- Create, edit, and delete records  
- Modal-based forms for create & edit  
- Reusable and modular components  
- State lifting and basic state management  
- Data fetching using `useEffect`

---

## Tech Stack

- React  
- JavaScript (ES6+)  
- CSS (minimal, intentional)
